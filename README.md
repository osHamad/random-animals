# Random Animals
A web aplication that uses APIs to generate random animal images. There are several species that can be generated, including:

1. Cats
2. Dogs
3. Foxes


## Usage
### Requirements
Node.js Required

### Dependencies
Express.js is required. Run the following.

`$ npm install express`

### Run
Standard run:

`$ node server.js`

Or run with Nodemon

`$ npm install nodemon`

`$ npm run start`
